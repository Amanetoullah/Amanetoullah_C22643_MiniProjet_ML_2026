## Présentation du projet
Ce projet porte sur l'implémentation et l'analyse de deux modèles fondamentaux de l'apprentissage supervisé : la Régression Linéaire Multiple et la Régression Logistique. L'étude s'appuie sur des jeux de données réels pour illustrer les capacités de prédiction et de classification de ces algorithmes.

## Algorithmes implémentés

### Apprentissage Supervisé (MLS)
- **Régression Linéaire :** Ce modèle est utilisé pour prédire les charges financières des assurés. L'algorithme cherche à établir une relation linéaire entre des variables explicatives (âge, IMC, statut de fumeur) et une variable cible numérique.
- **Régression Logistique :** Ce modèle est appliqué à une problématique de classification binaire. Il utilise la fonction sigmoïde pour estimer la probabilité d'appartenance d'une observation à une classe donnée.


## Méthodologie
1. Exploration des données et analyse des corrélations.
2. Prétraitement : Nettoyage des données et encodage des variables catégorielles.
3. Normalisation des données pour la régression logistique.
4. Entraînement, évaluation des performances et interprétation des coefficients.
